Dev Notes — Agencies List (agencies.html)
  • Route: /agencies — public, no auth required.
  • Data: GET /api/public/agencies?hasActiveJobs=true — returns agencies with at least one active job, ordered by active job count descending. Each record includes name, city, state, logoUrl, description, and activeJobCount.
  • Search filters client-side on name and city, or server-side via ?q= param for SEO-friendly URLs (e.g. /agencies?q=austin).
  • Description is truncated to 3 lines via CSS -webkit-line-clamp — no server-side truncation needed.
  • Logo falls back to initials avatar if logoUrl is null or 404.
  • Each "View Agency →" links to /agencies/{orgSlug}.

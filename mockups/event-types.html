How Types Are Used

Types appear in the Event Type dropdown when adding or editing an important date. The color you assign here is used as a visual tag on calendar views and the candidate timeline. Inactive types are hidden from that dropdown but historical records are preserved.

📋 Dev Notes:
  • Types are per-department (scoped to the logged-in org) — different clients have different type lists.
  • Deleting a type should be blocked (or soft-delete only) if any existing Event records reference it — show an error explaining why.
  • Color is stored as a hex string. The color dot appears in the Add Event form next to each type option and on calendar/timeline views.
  • The "Inactive" state hides the type from new-entry dropdowns while preserving all historical records that used it.
  • Clicking "Edit" should populate the left form with that row's values and change the Save button label to "Update Event Type".

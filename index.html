Feature mockups & wireframes for developer handoff

AllStarRecruiter.com — Homepage NEW
Main marketing site for allstarrecruiter.com. Full product pitch, pipeline UI mockup, proof stats, testimonials, "backed by All-Star Talent" section. No free tier.

Job Board — Listings & Search NEW
Public-facing job board. Hero search, category/location/salary filters, job cards, pagination.
Job Detail — Full Posting NEW
Full job description, requirements, benefits, apply sidebar with inline form, similar jobs.
Job Detail — Branded Page NEW
Branded job detail experience built by the agency in the web builder. Department hero, values, photos, benefits, dual-track requirements, apply CTA.
Job Application Form NEW
Standalone apply page. Name, email, phone, position, message, email/SMS opt-ins. Confirmation state on submit.
Agencies — Search & Browse NEW
Public-facing agency directory. Search by name or city, grid of agency cards with description and open job count.
Agency Profile — Public Page NEW
Candidate-facing agency profile. Agency description, open job listings, agency info sidebar.

Repositioned as a Guardian partner channel offer. Guardian Partner Free Plan, built-from-real-campaigns section, workflow, services upsell, pain-based Pro framing, referral tracking, UTM params.

Targets Police Chiefs' Associations. Free tier includes Web Builder, Email, Tagging & LE Job Board. Pro upsell with SMS, Events, Campaigns, AI and more.

Job Postings — Management List NEW
CRM jobs dashboard. Stat cards, filter/search, table of active/draft/closed postings with candidate counts.
Post a Job — Create / Edit Form NEW
Full job posting form. Details, location, compensation, rich-text description, publishing sidebar with listing page selector.
Agency Settings — Profile & Page Link NEW
CRM settings page. Agency logo, abbreviation, avatar color, about description, web builder or external URL link.

Tag Views — Organized, Timeline & Form NEW
Three toggleable tag pane layouts on a candidate detail view. Organized (grouped pills + favorites + hidden), Timeline (chronological with dates), Form/CRM (radio + checkbox fields with date editing).

Events List — Calendar Landing Page NEW

Stat cards, filter bar, bulk actions, paginated roster. Designed for 247+ invitees.
Event Types — Settings Page NEW
Manage custom event types (In-Person, Interview, Virtual…) with colors. Matches current functionality.
